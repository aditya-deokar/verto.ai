## ✅ Pre-Interview Checklist

### 🖥️ Technical Setup
- [ ] Local dev server starts without errors (`bun run dev`)
- [ ] Database has at least 2-3 sample presentations
- [ ] One pre-generated presentation ready as demo backup
- [ ] Browser bookmarks: editor page, create page, landing page
- [ ] Terminal ready with project open
- [ ] VS Code / editor open with key files bookmarked

### 🧠 Knowledge Check
- [ ] Can explain the project in 30 seconds without hesitation
- [ ] Can name all 8 agents in the pipeline
- [ ] Can explain the layout-before-content innovation
- [ ] Can discuss 3 trade-offs with rationale
- [ ] Can answer "what would you improve?" with 3 specific answers
- [ ] Can draw the architecture on a whiteboard in 60 seconds

### 🎤 Delivery Practice
- [ ] Practiced 2-minute pitch at least twice
- [ ] Ran through the demo flow at least once
- [ ] Rehearsed answers to "tell me about a complex challenge you solved"
- [ ] Prepared your "Why did you build this?" story

---

## 🔑 The One Rule

> **Lead with the problem, not the technology.**
>
> Don't say: *"I built a Next.js app with LangGraph and Gemini."*
>
> Say: *"I noticed that creating presentations is tedious and AI tools just dump text into slides without understanding visual design. So I built a system that uses 8 specialized AI agents to generate layout-aware, visually polished presentations in under 60 seconds."*

*Good luck! You've built something genuinely impressive. Now go show them why.* 🚀
